# Notebook Knight

Notebook Knight is a mobile-first hand-drawn JRPG built around the idea of a fantasy adventure doodled on loose-leaf school paper.

## Current Production Status

- Game mechanics: Alpha Locked
- Code/gameplay: Build 22 current master
- Current phase: Art Production — Phase 1 UI
- Approved UI assets:
  - `ui_main_menu` — Main Menu v1.0

## Folder Guide

- `00_Project_Management/` — Production documents, checklists, schedules
- `01_Art/` — Approved, WIP, and rejected art assets
- `02_Game_Builds/` — HTML builds and archived versions
- `03_Audio/` — Music and sound effects
- `04_Playtesting/` — Feedback and QA reports
- `05_Releases/` — Release candidates and final builds

## Asset Naming

Use lowercase snake_case.

Examples:

- `ui_main_menu_v1.png`
- `enemy_goblin_v1.png`
- `weapon_sword_of_light_v1.png`
- `fx_fireball_v1.png`

## Production Rule

Approved assets should not be replaced unless a genuine issue is discovered.
